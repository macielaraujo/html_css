Citações

Para citações: <q> -> Esta é uma citação simples

Para citações complexas: <blockquote>

Esse é um exemplo de como o Texto da citação ficaria ao ser usado o blockquote, atente-se para o alinhamento do texto e para o parâmetro cite da tag que visualmente não muda nada, mas servirá para os mecanismos de buscas localizar o autor em questão, podemos passar uma URL que leve ao autor ou para a obra citada.

Abreviação

A tag <abbr> será útil para o entendimento de abreviações
Por exemplo, HTML e CSS

Texto Invertido

Não parece muito util, mas temos a tag <bdo> que serve para inverter o texto. Ex.: Assim ficará o texto depois de definir a direção do texto, se ltr ou rtl.
